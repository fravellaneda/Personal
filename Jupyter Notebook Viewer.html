Regresion Lineal

Introduccin

La Regresion Lineal es un modelo matemtico usado para aproximar la relacin entre una variable dependiente YY y m variables dependientes X1,X2,...,XmX1,X2,...,Xm. Este mdelo se puede expresar de la siguiente forma:

Y=0+1X1+...+mXm(1)(1)Y=0+1X1+...+mXm

En el caso ms sencillo, es decir, cuando tenemos solo una variable independiente XX, la regresin lineal puede entenderse como una recta

y=1x+0y=1x+0

que aproxima linealmente de mejor manera la relacin de los datos. A continuacin se muestra una imagen en la cual podemos evidenciar esta aproximacin.

La primera regresin se hizo utilizando el metodo de los minimos cuadrados y fue publicada por Legendere en 1805. Aunque ms tarde gauss public un trabajo en donde desarrollaba de manera ms profunda el mtodo de los mnimos cuadrados.

El termino regresin se utiliz por primera vez en el estudio de variables antropomtricas al comparar las estaturas de los padres e hijos. En donde encontraron que los valores de las estaturas de los hijos con respecto a la media regresaban al promedio. Por est razn empez a utilizarse el trmino regresin.

Planteamiento del algortimo

El problema de la regresin lineal consiste en elegir unos valores determinados para los parametros jj de modo que la ecuacin (1) quede totalmente determinada. Para ello, es necesario un conjunto de datos y un metodo de aproximacin que usualmente es el mtodo de los minimos cuadrados. El cual, busca encontrar los coeficientes jj que minimizan el cuadrado de la distancia de las observaciones al modelo. Es decir, supongamos que tenemos nn observaciones y mm variables independientes, en este caso estamos interesados en hallar los coeficientes jj que minimizan la siguiente ecuacin.

def loss(b0,b1,x,y):
    return ((y-(b1*x+b0))**2).mean()

## Diferenciacin Automtica
grad_loss=jit(grad(loss,argnums=(0,1)))

Adems vamos a utilizar el mtodo del gradiente descendente para eso definimos las siguientes funciones:

In [214]:
def optimize(b0,b1,x,y,h=0.01):
    
    # Calcular Derivada
    b0_grad,b1_grad=grad_loss(b0,b1,x,y)
    
    # Gradiente Descendiente
    b1=b1-b1_grad*h
    b0=b0-b0_grad*h
    
    # Regresar Nuevos parmetros
    return b0,b1

Conclusiones

  • La regresin lineal es muy buena herramienta para predecir valores numericos para ciertos conjuntos de datos. Aunque es importante tener en cuenta que si bien las aproximaciones que brinda esta no son perfectas si constituye una manera de entender y predecir el funcionamiento de ciertos conjuntos de datos.

  • Fueron necesarias epocas y un una aproximacin inicial cercana a (0,0)(0,0) para que el algoritmo convergiera a un valor. Esto se debe a que la forma de los datos no es precisamente una linea, por eso creo que el algoritmo no es tan rapido y el ajuste no es perfecto.

  • Si bien ya habia realizado anteriormente modelos usando la regresion lineal, decid hacer este algoritmo a mano y utilizando solamente una variable ya que siempre perdia el referente grafico al realizarlo en varias dimensiones. Por esta razn, queria hacer un ejemplo sencillo de una sola variable para poder evidenciar el comportamiento y entender un poco mejor la funcin de prueba. Con esto entiendo un poco ms como funciona el modelo y siento que ya puedo llevarlo mucho mejor a varias dimensiones.
